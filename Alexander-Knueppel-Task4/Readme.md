# SPL - Machine Learning Toolbox, Ex. 4 (Blac-Box-Framework)

Plugins lassen sich in der Main-methode (application/App.java) einbinden bzw. auskommentieren.

Bisherige Plugins sind:

* IB1 und ZeroR als Klassifizierer
* Kappa Statistik und Genauigkeit als Qualitätsmessungen
* Arff und CSV als file extensions (CSV ist leer implementiert, leider zu wenig Zeit diese Woche)

Als parameter muss --gui für die GUI oder --cli für die Kommandozeile angegeben werden.
