# SPL - Machine Learning Toolbox, Ex. 3 (Antenna)

## Verfügbare Argumente

* --cli: Starte command line interface.
* --gui: Starte GUI

Eines dieser Argumente ist nur verfügbar/notwendig, falls eine Variante gewählt wird, die sowohl CLI als auch GUI als Feature beinhaltet.


## Preprocessor-gestützte Features

Die folgenden Features können (unter umständen) an- bzw- abgewählt werden:

* Graphical bzw. Console: entscheidet, welche Benutzerschnittstelle integriert werden soll
* IB1: zweiter (und erster sinnvoller) Klassifizierungsalgorithmus
* Debug: Debug-Modus aktiviert logging
* ConsoleLogger bzw. FileLogger: entscheidet, wohin geloggt werden soll
* Histogram: ist GUI aktiviert, kann von den geladenen Daten ein Histogram angezeigt werden.
* Arff: Dateiformat (da es das einzige ist, muss es in jeder Kombination ausgewählt werden)
* KappaStatistics: misst Qualität des Klassifizierungsalgorithmus'
